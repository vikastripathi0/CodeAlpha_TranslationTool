# 🌍 Language Translation Tool

A simple web app that translates text between multiple languages, built with Python and Streamlit.

## Features
- Translate text between 10 languages (English, Hindi, French, Spanish, German, Japanese, Chinese, Arabic, Russian, Portuguese)
- Clean, simple web interface — no coding needed to use it
- Instant translation powered by Google Translate

## Tech Stack
- Python
- [Streamlit](https://streamlit.io/) — for the web interface
- [deep-translator](https://pypi.org/project/deep-translator/) — for translation

## How to Run Locally

1. Clone this repository:
   ```
   git clone https://github.com/<your-username>/CodeAlpha_TranslationTool.git
   cd CodeAlpha_TranslationTool
   ```

2. Install dependencies:
   ```
   pip install streamlit deep-translator
   ```

3. Run the app:
   ```
   streamlit run app.py
   ```

4. Open the local URL shown in your terminal (usually `http://localhost:8501`)
